Print semua page ke PDF

Print setting:
1. Destination: Save as PDF
2. Layout: Landscape
3. Paper Size: A4
4. margin: none
5. option: background graphics - yes
		
Untuk ubah tanda tangan, copy image tanda tangan yang diinginkan ke folder 'resources' dan beri nama 'tanda-tangan.png' (image tanda-tangan.png yang lama dihapus dulu)

Gambar tanda tangan disarankan dengan background transparan agar tidak terlihat blok gambar dengan background
